h1. Check

h2. Descripción

Permite definir un check box

h2. Parámetros

* **name** - (opcional) - Identificador del elemento
* **titulo** - Etiqueta que se mostrará al lado del check, si no se especifica se mostrara el nombre del campo
* **campo** - Atributo donde se almacenará el valor del check
* **anchoTitulo** (opcional) - Permite indicar el ancho de la etiqueta de titulo.

h2. Definición

bc. Check:
	'Check' (name=ID)? (
		  ('campo' '=' campo=STRING)
		& ('titulo' '=' titulo=STRING)?	
	)
;	

h2. Ejemplo de uso

bc. Check titulo="Estoy de acuerdo" campo="solicitud.aceptacion.declaraciones"
