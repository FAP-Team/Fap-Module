h1. Editar Archivo

h2. Descripción

Elemento que permite modificar los campos tipo y descripción de un documento previamente subido al sistema de archivos que utilicemos.

h2. Parámetros

* **_[name]_** : Identificador del elemento.
* **campo = _[campo]_** : Especifica la entidad de la que se va a obtener la uri, descripción y tipo del documento. La uri se necesita para recuperar el archivo previamente subido al sistema de archivos que utilicemos, mientras que la descripción y el tipo serán para actualizar las propiedades del documento.
* **requerido** (opcional) : Indica la obligatoriedad de la edición de un documento.

h2. Definición

bc. EditarArchivo:
	elemento='EditarArchivo' name=ID (
		('campo' '=' campo=Campo)
		& (requerido?='requerido')?
	)
;

h2. Ejemplos de uso

h3. Elemento EditarArchivo dentro de un Form

bc. Popup PopUpDocumentacion titulo="Documentación" campo=Solicitud.documentacion.documentos {
   EditarArchivo modificarDocumento campo=Documento requerido
}
