h1. Subir Archivo AED

h2. Descripción

Elemento para la subida de documentos al Archivo Electrónico de Documentos. El documento se crea sin clasificar, y se coloca en la ruta donde se van a almacenar los archivo temporales.

h2. Parámetros

* **_[name]_** : Identificador del elemento.
* **campo = _[campo]_** : Especifica la entidad en la que se va a guardar el tipo y la descripción del documento subido.
* **requerido** (opcional) : Indica la obligatoriedad de la subida de un documento.
* **tramite** (opcional) : Sirve para filtrar los documentos, por el tipo de trámite, que se pueden subir.
* **aportadoPor** (opcional) : Sirve para filtrar los documentos, que se pueden subir, por el tipo de persona que lo puede hacer (Por ejemplo puede haber ocasiones en que un determinado tipo de documento lo deba subir el Ciudadano y otro tipo especifico debe ser aportado por la Administración).

h2. Definición

bc. SubirArchivoAed:
	elemento='SubirArchivoAed' name=ID (
		('campo' '=' campo=Campo)
		& (requerido?='requerido')?
		& ('tramite' '=' tramite=STRING)?		
		& ('aportadoPor' '=' aportadoPor=STRING)?
	)
;

h2. Ejemplos de uso

h3. Elemento SubirArchivoAed dentro de una Pagina

bc.. Pagina Documentos campo=Solicitud.documentacion.documentos {
	Accion crear redirigir=SubirArchivoAedTest	
	SubirArchivoAed fileAportacion campo=Documento requerido tramite="Solicitud"
}
	
Pagina SubirArchivoAedTest campo=Solicitud {
	Tabla documentos campo=Solicitud.documentacion.documentos paginaCrear=Documentos{
		Columna campo=Documento.id
	}	
}
