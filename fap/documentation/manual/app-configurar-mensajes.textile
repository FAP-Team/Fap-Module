h1. Configuración de mensajes

En la versión actual, se ofrece la posibilidad de personalizar los mensajes de las diferentes páginas de la aplicación. Las páginas a las que se les haya configurado un mensaje aparacen en la tabla de la página "Configurar mensajes" del apartado de Administración.

!images/admin-configurar-mensajes!

A la hora de crear un nuevo mensaje habrá que rellenar una serie de campos. En primer lugar, se rellena la página en la que se quiere configurar el mensaje. A continuación, tenemos la posibilidad de elegir el tipo del mensaje:

* Fatal
* Error
* Warning
* Ok
* Info
* Wiki

En el cuadro de texto disponible escribimos el mensaje deseado para el tipo de mensaje elegido y obtenemos la visualización bajo el botón guardar.
Para que se aplique el cambio de mensaje en la página, debe activarse el check de habilitación.

!images/admin-rellenar-mensajes!
