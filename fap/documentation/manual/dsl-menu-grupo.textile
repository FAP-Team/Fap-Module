h1. Grupo menú

h2. Descripción

Permite añadir un grupo dentro del menú que ha sido especificado. Dentro de un grupo pueden haber enlaces o más grupos anidados.

h2. Parámetros

* **titulo** - Texto que se mostrará en la cabecera del grupo.
* **permiso** (opcional) - Nombre del permiso asociado al grupo.

h2. Definición

bc. MenuGrupo:
   'Grupo' titulo=STRING 
   (permiso?="permiso" "=" namePermiso=STRING)? '{'
      elementos +=MenuElemento*
   '}'
;

h2. Ejemplos de uso

h3. Menu con varios grupos

bc. Menu {
   
   Grupo "Solicitud" {
      Enlace "Solicitante" pagina="Solicitante"
      Enlace "Documentación" pagina="Documentacion"
   }
   
   Grupo "Instruccion" permiso="instruccion" {
      Enlace "Aportación de Documentacion" pagina="Aportacion"
      Enlace "Verificacion"
      Enlace "Seguimiento"
   }
   
}
