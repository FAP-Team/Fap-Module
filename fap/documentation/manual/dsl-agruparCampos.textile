h1. Agrupar campos

h2. Descripción

Permite agrupar elementos en disposición horizontal

h2. Parámetros

No tiene parámetros

h2. Definición

bc. AgruparCampos:
	'AgruparCampos' '{'
		elementos+=Elemento*
	'}'
;

h2. Ejemplo de uso

bc. AgruparCampos{
	Texto campo="solicitud.solicitante.fax" titulo="Fax"
	Texto campo="solicitud.solicitante.email" titulo="Correo electrónico" requerido
}
