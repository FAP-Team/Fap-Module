h1. Wiki

h2. Descripción

Convierte el texto escrito en formato MediaWiki, en html. El formato MediaWiki es el utilizado en los artículos de la wikipedia. Puedes consultar la sintaxis del formato en el siguiente enlace: <a href="http://www.mediawiki.org/wiki/Help:Formatting">Documentación MediaWiki</a>

h2. Parámetros

* **estilo** (opcional) : Especifica un estilo CSS para el wiki.

h2. Definición

bc. Wiki:
	'Wiki' ('estilo' '=' estilo=STRING)? '{'
		wikiData+=STRING+
	'}'
;

h2. Ejemplo de uso

bc. Wiki {
	"cadena 1"
	"cadena 2
		cadena 2.1
		cadena 2.2"
	"enlace externo http://www.google.es"
	"enlace interno [[solicitante]]"
}
