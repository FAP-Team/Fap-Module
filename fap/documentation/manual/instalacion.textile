h1. Instalación

Para crear un entorno para poder utilzar FAP necesita una serie de requerimientos previos, unos obligatorios y otros no. La instalación de Play Framework es un requisito indispensable; no así los demás pasos, pero si recomendables.

h2. Instalación de Play Framework

FAP en su versión 2.1 está construido sobre Play 1.2.4, así que deberías instalar esa versión de Play. Para instalarlo solamente sigue los pasos que se indican en "http://www.playframework.org/documentation/1.2.2/install":http://www.playframework.org/documentation/1.2.2/install

h2. Instalación de FAP

Tranquilo... FAP es un módulo de Play, así que no hace falta ningún paso adicional de configuración para instalarlo. Al crear un nuevo proyecto, diremos que queremos utilizar el módulo y Play se encargará de todo.

h2. Configuración Avanzada

Opcionalmente te recomendamos seguir estos pasos en la configuración, pero para nada son obligatorios.

h3. Configuración de Eclipse

Playframework se integra con varios IDEs. Puedes consultar cómo integrarlos en "http://www.playframework.org/documentation/1.2.4/ide":http://www.playframework.org/documentation/1.2.4/ide

Si tu elección es Eclipse, FAP-SDK incluye un plugin para ese IDE que permite resaltado de sintaxis para escribir el DSL. Para utilizarlo sigue "estos pasos":curso-plugin

h3. Instalación de la base de datos

Playframework se puede configurar para trabajar con múltiples bases de datos. Tendrás que instalar alguna de ellas a parte si quieres utilizarlas.

_NOTA_: En las aplicaciones creadas por el equipo utilizamos "MySQL":http://www.mysql.com/ y "HeidiSQL":http://www.heidisql.com/


h3. Configuración para Desarrolladores del módulo FAP

Si quieres ayudar a desarrollar el módulo, o simplemente quieres ver como funciona realmente internamente, deberás seguír los siguientes pasos.

h3. Instalación del FAP-SDK

* Crea una nueva variable de entorno con el nombre FAPSDK y el valor de la ruta donde descromprimió el SDK.
** Windows : Click derecho Equipo > Propiedades, Configuracion Avanzada del Sistema (en el lateral izquierdo), (Pestaña Opciones Avanzadas) Botón Variables de entorno. Se deberan crear/modificar dos variables de entorno: FAPSDK=C:\fap-sdk\ y modificar la variable de entorno "PATH" añadir la ruta del framework play! (p.e. ...;C:\fap-sdk\play-1.2.4;...).
** Mac/Linux: Editar ~/.bashrc y añadir   export FAPSDK=/ruta/a/fap-sdk/

* Entorno para modificar el generador:
** Eclipse con versión de xtext 2.0.1 ("Instalación":xtext-instalacion).
** "ant":http://ant.apache.org/ mediante línea de comandos.
